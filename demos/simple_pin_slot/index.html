Simple Pin-Slot Constraint

The minimal honest mechanism: a crank pin drives a horizontal slotted slider. The slider position is solved from the pin/slot constraint first, then the 3D solids are rendered from that valid state.

Controls

This has one input DOF: crank angle. The yoke has one allowed DOF: horizontal translation. The vertical slot is sized from pin radius + clearance; the slider position is solved so the pin center lies on the slot centerline.

Invalid geometry: pin reaches beyond a rounded slot end. Increase slot length or reduce crank radius.
