# Four-Bar CAD Pilot

A source-controlled CAD pilot generated with `earthtojake/text-to-cad` and Python/build123d.

- Source: `src/four_bar_linkage.py`
- Browser model: `assets/four_bar_linkage_pose_45deg.glb`
- Primary CAD: `assets/four_bar_linkage_pose_45deg.step`
- Mesh export: `assets/four_bar_linkage_pose_45deg.stl`
- Validation: `docs/inspect-four-bar.txt`

Open `index.html` through GitHub Pages for an interactive web view.
