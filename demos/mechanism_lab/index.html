Mechanism Lab

First local prototype: an Artobolevsky-inspired hybrid indexing mechanism. It combines a Geneva-style dwell/index behavior with a slotted yoke output, so one steady motor can create pause → fast index → controlled slide stroke.

Innovation angle

Useful for low-cost automation: pill sorting, label stepping, camera shutters, educational robots, micro-conveyors.
